#### Import Process
1. User clicks "Import" button
2. File picker opens (JSON files only)
3. JSON file is parsed
4. Extracts `addresses` array from JSON
5. Maps each address to wallet format
6. Adds imported wallets to existing wallets array
7. Preserves existing wallets, appends new ones

## Technical Implementation Details

### Component Architecture

```
ContactDetailsModal
├── WalletsTab
│   └── WalletAddressCard (multiple)
├── GPG Tab (placeholder)
├── Invoices Tab (placeholder)
└── Notes Tab (placeholder)
```

### Data Flow

1. **Wallet Import:**
   ```
   JSON File → handleFileImport() → Parse JSON → Map Fields → Add to wallets array → Save on form submit
   ```

2. **Wallet Display:**
   ```
   ContactDetailsModal → WalletsTab → WalletAddressCard
   ContactDetailsModal watches wallet count → Updates badge
   ```

3. **Wallet Deletion:**
   ```
   WalletAddressCard emits delete → WalletsTab handles → Calls deleteWallet() → Refreshes list → Emits to parent → Updates count
   ```

### CSS Architecture

#### Modal Scrolling Fix
```css
.modal-content {
  overflow: hidden; /* Prevents outer scroll */
  display: flex;
  flex-direction: column;
}

.profile-sidebar {
  position: sticky;
  top: 0;
  overflow-y: auto; /* Independent sidebar scroll */
}

.profile-body {
  overflow-y: auto; /* Independent content scroll */
}
```

#### Wallet Card Layout
```css
.wallet-item {
  aspect-ratio: 1; /* Square cards */

}

.wallet-network {
  position: absolute;
  top: 0.5rem;
  left: 0.5rem;
}

  position: absolute;
  top: 0.5rem;
  right: 0.5rem;
}
```

### Custom Dropdown Implementation

#### State Management
- `openDropdownIndex: ref<number | null>` - Tracks which dropdown is open
- Click outside handler with capture phase listener
- Prevents event bubbling with `@click.stop`

#### Dropdown Features
- Absolute positioning with z-index: 1000
- Max height with scroll for long lists
- Visual selection state
- Smooth transitions

## JSON Import Format

### Expected Structure
```json
{
  "addresses": [
    {
      "currency": "BTC",
      "address": "1A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa",
      "keyFingerprint": "ABC123...",
      "cryptoPublicKey": "...",
      "gpgPublicKey": "..."
    }
  ]
}
```

### Field Mapping
- `currency` → `coinTicker` (required)
- `address` → `address` (required)
- `keyFingerprint` → `keyFingerprint` (optional, displayed on card)
- `cryptoPublicKey` → `cryptoPublicKey` (optional, stored but not displayed)
- `gpgPublicKey` → `gpgPublicKey` (optional, stored but not displayed)
