### How CurrencyCore can be utilized

- **Unified crypto integrations**: Single TS interface to 100s of services for exchanges, DEXs, rates, oracles, staking, mining, domains.
- **Price discovery and routing**: Aggregate rates from CEX/DEX/quick exchanges and choose best route.
- **Wallets and portfolio apps**: Multi-chain address utils, domains, explorers, staking providers, onramps.
- **Trading and execution bots**: Cross-venue quotes, order/trade tracking, basic risk signals from oracles/aggregators.
- **Analytics and monitoring**: Explorer APIs + oracles for chain data, prices, alerts, dashboards.
- **Developer platforms**: Provide a simplified SDK across fragmented vendor APIs with consistent types and error handling.

### Monetization strategies

- **SaaS API access**: Hosted API with tiered quotas and feature gating by category (e.g., CEX/DEX/oracles/staking).
- **SDK licensing**: Commercial license for on-prem/private code usage (per-seat or per-organization), updates included.
- **Enterprise support**: SLAs, custom integrations, priority requests, compliance features (audit logging, IP allowlists).
- **Vertical bundles**: Sell focused bundles (e.g., “Pricing + DEX routing”, “Staking + Oracles”, “Domains + Explorer”).
- **Usage-based add-ons**: Per-request overages, premium endpoints (historical OHLCV, advanced routing, backfills).
- **Professional services**: Custom connectors, integration help, migration, security reviews.

- Key management and secrets handling: recommend Vault/KMS integrations.
