## üìä API Endpoints Included

### Public APIs (No Authentication)
- ViaBTC pool stats: `https://viabtc.com/api/v1/bch/pool_stats`
- F2Pool coin stats: `https://api.f2pool.com/bitcoin-cash`
- BTC.com pool stats: `https://bch.btc.com/api/pool/stats`
- Mining-Dutch stats: `https://www.mining-dutch.nl/api/status`

### Authenticated APIs
- AntPool account: `https://api.antpool.com/account`
- Poolin miner stats: `https://api.poolin.com/api/miner/{address}`
- Huobi Pool stats: `https://api.huobipool.com/v1/miner/stats`
- Foundry USA group: `https://api.foundryusapool.com/account/group`

### Price Data APIs
- CoinGecko: `https://api.coingecko.com/api/v3/simple/price?ids=bitcoin-cash&vs_currencies=usd`
- Binance: `https://api.binance.com/api/v3/ticker/price?symbol=BCHUSDT`
- Kraken: `https://api.kraken.com/0/public/Ticker?pair=BCHUSD`

## üí° Usage Examples

### Simple: Fetch Pool Statistics
```typescript
import { ViaBTCPool } from '@/components/currencyCore/miningPools/BCH.BitcoinCash';

const stats = await ViaBTCPool.fetchPoolStats();
console.log('Hashrate:', stats?.hashRate);
```

### Advanced: Calculate Mining Profitability
```typescript
import { ViaBTCPool, fetchBCHPrice, calculateUSDValue } from '@/components/currencyCore/miningPools/BCH.BitcoinCash';

// Get miner earnings
const minerStats = await ViaBTCPool.fetchMinerStats(address, apiKey);
const earningsBCH = minerStats?.earnings || 0;

// Get current BCH price
const bchPrice = await fetchBCHPrice();

// Calculate USD value
const earningsUSD = await calculateUSDValue(earningsBCH);

console.log(`Earnings: ${earningsBCH} BCH = $${earningsUSD.toFixed(2)} USD`);
```

### Expert: Aggregate Multiple Pools
```typescript

const aggregated = await aggregatePoolStats();
console.log('Pools:', aggregated.pools.length);
console.log('Total Hashrate:', aggregated.totalHashrate);
```


### HMAC-MD5 Signature (AntPool)
```typescript
const signature = crypto
  .createHmac('md5', secretKey)
  .update(paramString)
  .digest('hex')
  .toUpperCase();
```

### HMAC-SHA256 Signature (Huobi Pool)
```typescript
const signature = crypto
  .createHmac('sha256', secretKey)
  .update(payload)
  .digest('base64');
```
