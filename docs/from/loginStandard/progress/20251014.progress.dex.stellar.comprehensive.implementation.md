## üîß Technical Highlights

### Stellar's Unique Architecture

1. **Native DEX Protocol**
   - DEX functionality is **built into the core protocol**, not external smart contracts
   - All trading operations are protocol-level, ensuring maximum security and efficiency
   - Supports both **orderbook** (limit/market orders) and **AMM liquidity pools**

2. **Path Payments**
   - Automatic multi-hop currency conversion in a single transaction
   - Example: `XLM ‚Üí BTC ‚Üí USDC ‚Üí EUR` with optimal pricing
   - Enables seamless cross-asset trades

3. **Federated Byzantine Agreement (FBA)**
   - **3-5 second finality** for all transactions
   - No mining or traditional block production
   - Energy-efficient consensus mechanism

4. **Extremely Low Fees**
   - **0.00001 XLM per operation** (~$0.0000012)
   - One of the lowest fee structures in blockchain
   - Fixed fee regardless of transaction size

5. **Soroban Smart Contracts**
   - WebAssembly-based smart contracts
   - Written in Rust
   - Enables advanced DeFi applications
   - Future-proof for complex financial primitives

6. **Trustlines**
   - Users must add a trustline before receiving non-XLM assets
   - 0.5 XLM reserve per trustline
   - Protects against spam and unauthorized asset distribution
