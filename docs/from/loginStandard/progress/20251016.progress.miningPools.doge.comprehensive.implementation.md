## Files Created

### 1. Individual Pool Integrations (10 files)

#### **prohashing.ts** (~3.4KB)
- Multi-algorithm pool with comprehensive API
- API key authentication (query parameter)
- Functions: `fetchAccountStats`, `fetchCoinStats`, `fetchWorkerStats`
- Special features: Profit-switching, multi-coin payouts

#### **f2Pool.ts** (~3.2KB)
- One of the oldest and largest pools (est. 2013)
- Username-based authentication (no key required)
- Functions: `fetchCoinStats`, `fetchUserStats`, `fetchWorkerStats`
- Merged mining with Litecoin

#### **viaBtc.ts** (~3.8KB)
- Comprehensive multi-currency pool
- HMAC-SHA256 signature authentication
- Functions: `fetchPoolStats`, `fetchUserStats`, `generateSignature`
- REST API with WebSocket support

#### **litecoinPool.ts** (~3.5KB)
- Largest merged mining pool for LTC and DOGE
- API key authentication
- Functions: `fetchUserStats`, `fetchWorkerStats`, `fetchPoolStats`
- Simultaneous LTC and DOGE rewards

#### **aikaPool.ts** (~2.8KB)
- Dedicated Dogecoin mining pool
- Address-based (no authentication)
- Functions: `fetchPoolStats`, `fetchUserStats`
- Stratum configuration included

#### **poolin.ts** (~3.6KB)
- Multi-currency pool with DOGE support
- Header-based API key authentication
- Functions: `fetchPoolStats`, `fetchUserStats`, `fetchWorkerStats`
- Modern interface with detailed API

#### **zergPool.ts** (~3.4KB)
- Auto-switching pool for maximum profitability
- Wallet address-based API
- Functions: `fetchWalletStats`, `fetchMultiWalletStats`, `fetchNetworkStats`
- No registration required

#### **twoMiners.ts** (~3.8KB)
- Modern multi-coin pool with comprehensive API
- Public API (no authentication)
- Functions: `fetchPoolStats`, `fetchAddressStats`, `fetchPayments`, `fetchBlocks`
- Active community support

#### **miningPoolHub.ts** (~3.9KB)
- Multi-coin platform with auto-exchange
- HMAC-SHA256 signature authentication
- Functions: `generateSignature`, `fetchUserBalance`, `fetchUserStatus`
- Auto-exchange and profit-switching

#### **multipool.ts** (~2.7KB)
- Multi-coin pool with auto-switching
- Address-based authentication
- Functions: `fetchPoolStats`, `fetchUserStats`, `fetchWorkerStats`
- Scrypt algorithm support

## Usage Examples

### Basic Pool Query
```typescript
import { F2Pool } from '@/components/currencyCore/miningPools/DOGE.Dogecoin/f2Pool';

const coinStats = await F2Pool.fetchCoinStats();
console.log('Pool hashrate:', coinStats.hashrate);
```

### With Authentication
```typescript
import { ProhashingPool } from '@/components/currencyCore/miningPools/DOGE.Dogecoin/prohashing';

const apiKey = process.env.PROHASHING_API_KEY;
const accountStats = await ProhashingPool.fetchAccountStats(apiKey);
console.log('Your hashrate:', accountStats.hashrate);
```

### Price Calculation
```typescript
import { getDOGEPrice, calculateDogeValue } from '@/components/currencyCore/miningPools/DOGE.Dogecoin';

const price = await getDOGEPrice();
const usdValue = await calculateDogeValue(1000);
console.log(`1000 DOGE = $${usdValue.toFixed(2)}`);
```

### Merged Mining
```typescript
import { LitecoinPool } from '@/components/currencyCore/miningPools/DOGE.Dogecoin/litecoinPool';

const apiKey = process.env.LITECOINPOOL_API_KEY;
const stats = await LitecoinPool.fetchUserStats(apiKey);
console.log('LTC rewards:', stats.user.confirmed_rewards);
console.log('DOGE rewards:', stats.user.doge_confirmed_rewards);
```
