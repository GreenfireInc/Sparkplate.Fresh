## Technical Implementation Notes

### Band Protocol Integration
- Uses `@bandprotocol/bandchain.js` SDK
- gRPC endpoint: `https://laozi-mainnet3.bandchain.org/grpc-web`
- REST API: `https://api.bandchain.org`
- Oracle Script ID 1 for standard price feeds
- Supports custom oracle scripts
- Full IBC integration for cross-chain data

### Pyth Network Integration
- Uses `@pythnetwork/hermes-client` for off-chain access
- CosmWasm contracts for on-chain integration
- Hermes endpoint: `https://hermes.pyth.network`
- Pull model with cryptographic proofs
- Available on Osmosis, Injective, Cronos, Kava, Evmos
- Sub-second updates with confidence intervals

### CosmJS Integration Pattern
```typescript
import { StargateClient } from '@cosmjs/stargate';
import { CosmWasmClient } from '@cosmjs/cosmwasm-stargate';

// For general Cosmos SDK interactions
const client = await StargateClient.connect('https://rpc.cosmos.network');

// For CosmWasm oracle contracts
const wasmClient = await CosmWasmClient.connect('https://rpc.osmosis.zone');
```
