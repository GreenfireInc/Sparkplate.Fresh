## Mining Pools Implemented

### 1. Litecoinpool.org (litecoinpool.ts)
- **Type**: PPS (Pay Per Share)
- **Fee**: 0% (donations accepted)
- **Features**: Oldest and most trusted LTC pool, merged mining (LTC + DOGE), TLS security
- **API Endpoints**: User stats (requires API key)
- **Integration**: Complete API implementation with authentication support

### 2. F2Pool (f2Pool.ts)
- **Type**: PPS
- **Fee**: 2.5%
- **Features**: Large multi-currency pool, merged mining with DOGE
- **API Endpoints**: Coin stats, user stats (auth), worker stats (auth)
- **Integration**: Full API coverage with bearer token authentication

### 3. ViaBTC (viaBtc.ts)
- **Type**: FPPS / PPLNS
- **Fee**: 2% (FPPS), 1% (PPLNS)
- **Features**: Flexible payout options, global operations
- **API Endpoints**: Pool stats, miner stats (auth), block stats
- **Integration**: Comprehensive API with flexible payout support

### 4. Prohashing (prohashing.ts)
- **Type**: PPS
- **Fee**: 4.99%
- **Features**: U.S.-based, payouts in any currency (BTC, LTC, USD)
- **API Endpoints**: Account stats, coin stats, worker stats (all require auth)
- **Integration**: Multi-currency payout support with API key auth

### 5. Poolin (poolin.ts)
- **Type**: PPS+ / FPPS / PPLNS
- **Fee**: 2.5% (varies by mode)
- **Features**: Multiple payout modes, comprehensive mining tools, merged mining
- **API Endpoints**: Pool stats, miner stats (auth), worker stats (auth)
- **Integration**: Full API with bearer token authentication

### 6. AntPool (antPool.ts)
- **Type**: PPS+
- **Fee**: 2.5%
- **Features**: Major pool by Bitmain, merged mining (LTC + DOGE)
- **API Endpoints**: Account, hashrate, workers, payments (HMAC-MD5 auth)
- **Authentication**: HMAC-MD5 signature generation required
- **Note**: Complex authentication implementation documented

### 7. 2Miners (twoMiners.ts)
- **Type**: SOLO / PPLNS
- **Fee**: 1%
- **Features**: Transparent, beginner-friendly, no authentication required
- **API Endpoints**: Pool stats, miner stats, payments, blocks
- **Integration**: Public API with no authentication needed

## Technical Specifications

### LTC Network Details
- **Consensus**: Scrypt Proof of Work
- **Decimals**: 8 (same as Bitcoin)
- **Address Formats**: 
  - P2PKH (Legacy): Starts with 'L'
  - P2SH (SegWit Compatible): Starts with 'M'
  - P2WPKH (Native SegWit): Starts with 'ltc1'
  - MWEB: Starts with 'ltc1mweb'
- **Block Time**: ~2.5 minutes
- **Total Supply**: 84 million LTC
- **Merged Mining**: Dogecoin (DOGE) on most major pools

### API Authentication Methods
- **Public APIs**: 2Miners
- **API Key (URL Parameter)**: Litecoinpool.org, Prohashing
- **Bearer Token**: F2Pool, ViaBTC, Poolin
- **HMAC-MD5**: AntPool (complex signature generation)

### Stratum Configuration
Each pool includes stratum server details:
- Server hosts and ports
- TLS support where available
- Connection strings
- Regional server options

### Merged Mining Support
Major pools support simultaneous mining of:
- **Litecoin (LTC)** - Primary reward
- **Dogecoin (DOGE)** - Bonus reward
- No additional computational effort required
