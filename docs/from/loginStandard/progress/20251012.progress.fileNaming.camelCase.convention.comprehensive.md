## Technical Implementation Details

### Renaming Strategy
```bash
# Command pattern used for all renames:
mv src/components/[path]/PascalCaseFile.tsx src/components/[path]/camelCaseFile.tsx
```

### Import Update Pattern
**Before:**
```typescript
export { ComponentName } from './PascalCaseFile';
import { ComponentName } from './PascalCaseFile';
```

**After:**
```typescript
export { ComponentName } from './camelCaseFile';
import { ComponentName } from './camelCaseFile';
```

### Key Principle
- **File names:** camelCase (e.g., `userCard.tsx`)
- **Component names:** PascalCase (e.g., `export const UserCard`)
- **Exports:** Unchanged, still use PascalCase component names

## Naming Convention Standards

### File Naming Rules (Now Enforced)
```typescript
// ✅ CORRECT: camelCase for files
loginStandard.tsx
userCard.tsx
serverSelection.tsx
keystoreDecryptionModal.tsx

// ❌ INCORRECT: PascalCase for files
LoginStandard.tsx  // Old pattern
UserCard.tsx       // Old pattern
ServerSelection.tsx // Old pattern
```

### Component Export Pattern
```typescript
// File: userCard.tsx
export const UserCard = ({ name, isSelected, onClick }: UserCardProps) => {
  // Component implementation
};

// Still exported as PascalCase!
export default UserCard;
```

### Import Pattern
```typescript
// ✅ Correct import after refactoring
import { UserCard } from './userCard';
import { LoginStandard } from './loginStandard';

// Component names remain PascalCase
<UserCard name="Guest" />
<LoginStandard />
```

## Migration Notes for Future Reference

### If Adding New Components
```typescript
// ✅ DO: Use camelCase for file names
// File: src/components/newFeature/userProfile.tsx
export const UserProfile = () => { /* ... */ };

// ❌ DON'T: Use PascalCase for file names
// File: src/components/newFeature/UserProfile.tsx
```

### Pattern to Follow
1. **Create file:** `camelCaseFileName.tsx`
2. **Export component:** `export const PascalCaseComponent`
3. **Update index:** Add to barrel export if needed
4. **Import elsewhere:** `import { PascalCaseComponent } from './camelCaseFileName'`
