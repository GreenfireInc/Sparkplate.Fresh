## üîÑ Recommended Usage Pattern

### **Production Fallback Chain:**
```typescript
import {
  dogechainMainnet,  // Primary: Established, reliable
  blockchairDoge,    // Fallback 1: Fast, high limits
  electrsDoge,       // Fallback 2: Modern, open-source
  dogeClient         // Fallback 3: Professional infrastructure
} from '@/components/currencyCore/blockchainAPIs/DOGE.Dogecoin';

async function getBalanceWithFallback(address: string): Promise<number> {
  const apis = [dogechainMainnet, blockchairDoge, electrsDoge, dogeClient];
  
  for (const api of apis) {
    try {
      const result = await api.getBalance(address);
      return result.balance;
    } catch (error) {
      console.warn(`API ${api.constructor.name} failed, trying next...`);
      continue;
    }
  }
  
  throw new Error('All Dogecoin APIs failed');
}
```

## üí∞ Dogecoin-Specific Implementation Details

### **Unit Conversion**
- **Shibes (smallest unit)**: 1 DOGE = 100,000,000 shibes
- Similar to Bitcoin's satoshi system
- All internal calculations use shibes for precision

### **Transaction Fees**
- Default fee recommendations:
  - Fast: 1000 shibes/byte
  - Medium: 500 shibes/byte
  - Slow: 100 shibes/byte
- Typical transaction: ~1 DOGE total fee
- Much lower than Bitcoin fees

### **Block Time**
- Average: ~1 minute (vs Bitcoin's ~10 minutes)
- Faster confirmations than Bitcoin
- More frequent block rewards

### **Address Formats**
- **P2PKH**: Starts with 'D' (most common)
- **P2SH**: Starts with '9' or 'A'
- Example: `D7P2qjEuqVn7tkS9vFVV7Jq3a8Wb6hLDdv`

### **Supply Model**
- No maximum supply (inflationary)
- Block reward: 10,000 DOGE per block
- Annual inflation decreases over time
