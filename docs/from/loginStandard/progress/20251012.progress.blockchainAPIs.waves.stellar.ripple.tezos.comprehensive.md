- **Smart Accounts:** Accounts with custom scripts
- **Leasing:** Stake WAVES to nodes for rewards
- **Data Storage:** Key-value storage on-chain

## Technical Implementation Details

### Architecture Patterns

All API implementations follow consistent patterns:

1. **Configuration Interface:**
   ```typescript
   interface APIConfig {
     baseURL?: string;
     apiKey?: string;  // if required
     timeout?: number;
     retries?: number;
     network?: 'mainnet' | 'testnet';
   }
   ```

2. **Class-Based API Structure:**
   - Private axios client instance
   - Configuration management
   - Public methods for common operations
   - Error handling with meaningful messages

3. **Export Patterns:**
   - Singleton instances for free-tier APIs
   - Factory functions for configurable instances
   - TypeScript interfaces for all data structures

### Common Methods Across All APIs

- `getAccount(address)` - Account information
- `getBalance(address)` - Balance queries
- `getTransactions(address, limit)` - Transaction history
- `getTransaction(hash)` - Single transaction lookup
- `getBlock(height)` - Block information
- `submitTransaction(signedTx)` - Transaction broadcasting
