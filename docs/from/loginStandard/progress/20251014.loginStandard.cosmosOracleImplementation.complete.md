## Technical Implementation Notes

### IBC (Inter-Blockchain Communication)
- Oracle data flows between chains via IBC
- Band Protocol provides native IBC oracle feeds
- Pyth Network uses Wormhole + IBC
- Enables cross-chain price data without bridges

### CosmWasm Integration
- Oracle data accessible from smart contracts
- Band Protocol and Pyth Network have CosmWasm contracts
- Query oracle prices from within smart contracts

### ABCI++ Vote Extensions
- Cosmos SDK v0.50+ feature
- Validators submit oracle data during consensus
- Protocol-level security for oracle data
- Used by Cosmos SDK Oracle Module

### Pull vs Push Oracles
- **Pull (Pyth):** Users fetch price when needed, cost-effective
- **Push (Band, Chainlink, DIA):** Oracle updates pushed to chain regularly
- **Protocol-Level (Cosmos SDK):** Embedded in block production
