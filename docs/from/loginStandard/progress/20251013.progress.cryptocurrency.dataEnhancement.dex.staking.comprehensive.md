## üîß Technical Implementation Details

### Parameter Purposes

#### 1. **evmChainID**
- Identifies EVM-compatible chains for wallet integration
- Helps users understand blockchain architecture
- Critical for MetaMask and Web3 wallet connections
- Examples:
  - ETH: `1` (Ethereum mainnet)
  - BNB: `56` (BSC mainnet)
  - ETC: `61` (Ethereum Classic mainnet)
  - Non-EVM: `"N/A (UTXO-based chain)"` with explanation

#### 2. **typicalDerivationPath**
- Standard BIP44 or blockchain-specific paths
- Essential for HD wallet implementations
- Examples:
  - Standard EVM: `m/44'/60'/0'/0/0`
  - Bitcoin: Multiple paths for different address types
  - Substrate: `//polkadot//0` (different notation)
  - Waves: N/A (uses unique 15-word system)

#### 3. **sendingFunctionality**
- Documents SDK methods for transaction creation
- Helps developers implement sending features
- Examples:
  - Ethereum: `ethers.Wallet.sendTransaction()`
  - Solana: `@solana/web3.js Transaction with sendAndConfirmTransaction()`
  - Tezos: `@taquito/taquito Tezos.contract.transfer()`

#### 4. **dex (Array)**
- Up to 7 decentralized exchanges per currency
- Includes:
  - Native protocol DEXs
  - AMM DEXs
  - Aggregators
  - Cross-chain exchanges
  - Instant swap services (for UTXO chains)

#### 5. **stakingProviders (Array)**
- Up to 7 staking/delegation options
- Categories:
  - Native staking/delegation
  - Liquid staking protocols
  - Exchange staking
  - Institutional providers
  - Community validators
  - For PoW: "N/A" with explanation

#### 6. **miningPools**
- String for PoS chains: `"N/A (Proof of Stake - No Mining)"`
- Array of 7 mining pools for PoW chains
- Includes major pools with URLs and descriptions

## üåü Notable Blockchain Characteristics Documented

### Unique Consensus Mechanisms
- **Stacks (STX):** Proof of Transfer - earn BTC by locking STX
- **Stellar (XLM):** Stellar Consensus Protocol (Federated Byzantine Agreement)
- **XRP:** XRP Ledger Consensus - validators run pro bono
- **Tezos (XTZ):** Liquid Proof of Stake with "baking"
- **Algorand (ALGO):** Pure Proof of Stake
- **Waves (WAVES):** Leased Proof of Stake

### Special Features Highlighted
- **Litecoin/Dogecoin:** Merged mining capabilities
- **Stellar/XRP:** Built-in DEX at protocol level
- **Tezos:** Self-amending blockchain (no hard forks)
- **Waves:** 15-word seed phrase (not standard 12/24)
- **Tezos:** Coin type 1729 (Hardy-Ramanujan number)
- **Solana:** Proof of History + Proof of Stake hybrid

### Ecosystem Insights
- **Ethereum:** Largest DeFi ecosystem post-Merge
- **Solana:** Jupiter aggregator dominance
- **Cosmos:** IBC cross-chain with Osmosis DEX
- **Polkadot:** Parachain architecture with specialized DEXs
- **Stacks:** Bitcoin-native DeFi with Clarity language
