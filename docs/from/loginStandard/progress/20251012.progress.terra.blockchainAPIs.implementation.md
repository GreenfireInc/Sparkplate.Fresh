## APIs Implemented

### Terra 2.0 (LUNA) APIs - 6 Implementations

#### 1. Mintscan API (`mintscanAPI.ts`)
- **Provider**: Cosmostation
- **Base URL**: `https://api-terra.cosmostation.io`
- **Free Tier**: 10,000 requests/day, 2 requests/second
- **Key Features**:
  - Account balance and transaction history
  - Staking information and delegations
  - Validator data and rankings
  - Block queries
  - Rewards tracking
  - Transaction broadcasting
- **Singleton**: `mintscanAPI`

#### 2. Terra Finder API (`terraFinderAPI.ts`)
- **Provider**: Official Terra (LCD - Light Client Daemon)
- **Base URL**: `https://phoenix-lcd.terra.dev`
- **Free Tier**: Unlimited, no API key required
- **Key Features**:
  - Direct blockchain queries via REST
  - Account, transaction, and block data
  - Staking and governance information
  - Network parameters
  - Supply information
- **Singleton**: `terraFinderAPI`

#### 3. ATOMScan API for Terra 2.0 (`atomscanAPI.ts`)
- **Provider**: ATOMScan
- **Base URL**: `https://atomscan.com/terra2/api`
- **Free Tier**: Free with rate limits
- **Key Features**:
  - Account and transaction data
  - Price information
  - Transaction statistics
  - Validator information
  - Chain statistics
- **Singleton**: `atomscanLunaAPI`

#### 4. Stake.ID API (`stakeIdAPI.ts`)
- **Provider**: Stake.ID
- **Base URL**: `https://terra.stake.id/api`
- **Free Tier**: Free
- **Key Features**:
  - Search addresses, blocks, transactions, validators
  - Key statistics and metrics
  - Network visualization
  - Governance proposals
- **Singleton**: `stakeIdAPI`

#### 5. GetBlock API (`getblockAPI.ts`)
- **Provider**: GetBlock
- **Base URL**: Custom (requires API key)
- **Free Tier**: 50,000 compute units/month, 5 requests/second
- **Key Features**:
  - RPC and REST endpoints
  - Block, transaction, and account data
  - Network information
  - Transaction broadcasting
- **Factory Function**: `createGetBlockLunaAPI(apiKey, network?)`

#### 6. NOWNodes API (`nownodesAPI.ts`)
- **Provider**: NOWNodes
- **Base URLs**: 
  - RPC: `https://luna.nownodes.io`
  - REST: `https://luna-rest.nownodes.io`
- **Free Tier**: Free tier available (requires API key)
- **Key Features**:
  - Full node RPC and REST access
  - Block, transaction, and account data
  - Staking and governance queries
  - Network information
- **Factory Function**: `createNOWNodesLunaAPI(apiKey)`

### Terra Classic (LUNC) APIs - 5 Implementations

#### 7. ATOMScan API for Terra Classic (`terraClassicAtomScanAPI.ts`)
- **Provider**: ATOMScan
- **Base URL**: `https://atomscan.com/terra/api`
- **Free Tier**: Free with rate limits
- **Key Features**:
  - Terra Classic (LUNC) specific tracking
  - Account and transaction data
  - Historical data for original Terra chain
  - Validator information
  - LUNC price tracking
- **Singleton**: `terraClassicAtomScanAPI`

#### 8. LuncScan API (`luncScanAPI.ts`)
- **Provider**: LuncScan
- **Base URL**: `https://luncscan.com/api`
- **Free Tier**: Free with rate limits
- **Key Features**:
  - Track LUNC (native coin of original Terra blockchain)
  - **Burn tracking and statistics** (unique feature)
  - Account and transaction monitoring
  - Governance proposals
  - Historical burn data
- **Singleton**: `luncScanAPI`

#### 9. Terra Classic Tools API (`terraClassicToolsAPI.ts`)
- **Provider**: Terra Classic Tools
- **Base URL**: `https://api.terra-classic.money`
- **Free Tier**: Free
- **Key Features**:
  - Live LUNC and USTC price tracking
  - Burn monitoring
  - **USTC re-peg tracking** (unique feature)
  - Staking yields and DeFi information
  - Bridges and wallets information
  - Historical price and burn data
- **Singleton**: `terraClassicToolsAPI`

#### 10. NOWNodes API for Terra Classic (`terraClassicNOWNodesAPI.ts`)
- **Provider**: NOWNodes
- **Base URLs**:
  - RPC: `https://terra-classic.nownodes.io`
  - REST: `https://terra-classic-rest.nownodes.io`
- **Free Tier**: Free tier available (requires API key)
- **Key Features**:
  - Full node RPC and REST access for Terra Classic
  - Block, transaction, and account data
  - Staking and governance queries
  - **Terra Classic specific treasury and tax data** (unique feature)
- **Factory Function**: `createTerraClassicNOWNodesAPI(apiKey)`

#### 11. Terrasco.pe API (`terrascopeAPI.ts`)
- **Provider**: Terrasco.pe (Chainscope)
- **Base URL**: `https://terrasco.pe/api`
- **Free Tier**: Free with rate limits
- **Key Features**:
  - Terra Classic specific data
  - Account, transaction, and block information
  - Historical data and analytics
  - Validator performance metrics with uptime
  - Governance proposals and voting
  - Vesting account information
- **Singleton**: `terrascopeAPI`

### API Patterns
- **Singleton Pattern**: Used for public APIs (Mintscan, Terra Finder, ATOMScan, Stake.ID, LuncScan, Terra Classic Tools, Terrasco.pe)
- **Factory Pattern**: Used for APIs requiring keys (GetBlock, NOWNodes)
- **Dual Endpoint Support**: RPC and REST clients for NOWNodes implementations
- **Error Handling**: Comprehensive error handling for RPC calls
- **Type Safety**: Full TypeScript typing with no `any` types

### Network Support
- **Terra 2.0 (LUNA)**:

  - Native Token: LUNA
  - Address Prefix: `terra`
  
- **Terra Classic (LUNC)**:
  - Chain ID: `columbus-5` (mainnet)
  - Native Tokens: LUNC, USTC
  - Address Prefix: `terra`

## Usage Examples

### Terra 2.0 (LUNA) Example
```typescript
import { mintscanAPI, terraFinderAPI } from '@blockchainAPIs/LUNA.Terra';

// Get account balance
const balance = await mintscanAPI.getBalance('terra1...');
console.log('LUNA Balance:', balance);

// Get transaction history
const txs = await terraFinderAPI.getTransactionHistory('terra1...', 50);
console.log('Transactions:', txs);

// Get validators
const validators = await mintscanAPI.getValidators();
console.log('Active Validators:', validators);
```

### Terra Classic (LUNC) Example
```typescript
import { luncScanAPI, terraClassicToolsAPI } from '@blockchainAPIs/LUNA.Terra';

// Track burns
const burnInfo = await luncScanAPI.getBurnInfo();
console.log('Total LUNC Burned:', burnInfo.total_burned);
console.log('24h Burn:', burnInfo.burn_24h);

// Monitor USTC re-peg
const repegInfo = await terraClassicToolsAPI.getUSTCRepegInfo();
console.log('USTC Price:', repegInfo.current_price);
console.log('Re-peg Progress:', repegInfo.repeg_progress + '%');

// Get DeFi statistics
const defiStats = await terraClassicToolsAPI.getDefiStats();
console.log('Total Value Locked:', defiStats.total_value_locked);
```
