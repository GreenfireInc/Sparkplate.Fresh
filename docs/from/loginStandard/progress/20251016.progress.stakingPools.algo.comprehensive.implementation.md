## ALGO-Specific Considerations

- **Proof of Stake**: Algorand uses pure PoS (no mining)
- **Fast Finality**: 4.5 second block times
- **Low Fees**: ~0.001 ALGO per transaction
- **microAlgos**: 1 ALGO = 1,000,000 microAlgos (6 decimals)
- **Governance**: Quarterly periods with on-chain voting
- **Liquid Staking**: Multiple options (xALGO, tALGO, mALGO)
- **Smart Contracts**: TEAL/PyTeal language
- **Ed25519**: Cryptographic signature algorithm
- **25-word Mnemonic**: Standard seed phrase format
